Виджет "Погода"

Данное приложение (виджет) показывает текущую погоду в указанных городах. Информацию о погоде приложение получает с ресурса OpenWeather. Согласно рекомендации этого ресурса, запрос для конктретного города должен поступать не чаще чем 1 раз за 10 мин. Виджет обновляет данные о погоде при возврате в основное состояние из настроечного режима, а также при перезагрузке страницы.

При первом посещении автоматически включается режим настройки, и пользователю предлагается ввести название города, например Moscow, Saint Petersburg, New York, San Francisco и т.д. В случае успешной загрузки погодных данных для указанного города виджет переключается в основной режим. Последующие переключения между режимами осуществляются вручную. В случае опечатки в названии города на экране выводится сообщение "Город не найден". Если указанный город уже есть в списке, выводится сообщение "Город уже представлен", и отправка запроса становится невозможной.

Изменение порядка в списке городов происходит с помощью опции drag'n'drop, реализованной для экранов и настольных и мобильных устройств.

Конфигурационные настройки виджета хранятся в реактивной переменной приложения, а также сериализуются в localStorage по завершении работы. В начале следующего сеанса работы виджета данные загружаются из хранилища. Удалить все данные можно с помощью служебной кнопки Reset.
